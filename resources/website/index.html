De onderdelen:
Waar bestaat de pinautomaat uit?

Client

  • Pinpas scanner.
  • Scherm (pc).
  • Toetsen voor de invoer van de data.
  • Arduino.
  • USB kabel tussen Arduino en PC.
  • Bonnetjesprinter.

    • Server

  • Een systeem die op Linux draait.
  • Een database met de gebruikers en de rekening.
  • Opslaan van pin geschiedenis.
